Υπολογισμός Διακρίνουσας και επίλυση δευτεροβάθμιας εξίσωσης

Με την εύρεση της Διακρίνουσας (Δ) μπορούμε να υπολογίσουμε τις πιθανές τιμές του αγνώστου x μιας δευτεροβάθμιας εξίσωσης αx2+βx+γ=0.
Υπολογισμός διακρίνουσας: Δ=β2-4‧α‧γ
Λύσεις της εξίσωσης:
Εάν Δ > 0 Εάν Δ = 0 Εάν Δ < 0
(X 1,2) = (-β ± √Δ)/2α
H εξίσωση έχει 2 ρίζες (λύσεις) Χ1,2
(X 1,2) = -β/2α
H εξίσωση έχει 1 διπλή ρίζα (λύση) Χ1,2
Δεν υπάρχουν πραγματικές ρίζες (∈ R)

Παρακαλώ εισάγετε στα αντίστοιχα πεδία που ακολουθούν τις τιμές των α, β και γ.
